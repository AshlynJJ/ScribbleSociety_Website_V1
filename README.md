# ScribbleSociety_Website_V1
## Description
Scribble Society is a platform where artists yound and old, upcomming or experienced, display their art, sell it if they would like or take commissions from Masterpiece Collectors. It is a platform that offers guidance, no judgement. Art starts with a single scribble.

## How to run the program
Ensure you have Node.js (LTS version). If you do not then please download & install from https://nodejs.org</br>
Clone from the Git repo or download the zip file.</br>
Step 1:</br>
Open terminal, in Powershell enter these exact commands to clone the repo:</br>
git clone <repo-url>
cd "C:\path\to\ScribbleSociety_Website_V1"
</br>
Step 2: Install Dependecies</br>
run command in terminal: npm install </br>
Step 3: Start the server</br>
run command in terminal: node server.js</br>
